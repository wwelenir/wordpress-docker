# wordpress-docker

Wordpress base project in docker containing wordpress, mysql and adminer, all in the latest version

## Basic setup configuration
- Copy .env.example to .env
- Set your credentials values 

**Run docker compose**
  ```
  docker-compose up -d 
  ```
  or
  ```
  docker compose up -d 
  ```
  
**Wordpress url** 
  ```
  http://localhost:8030
  ```

**Adminier url** 
  ```
  http://localhost:8080
  ```
